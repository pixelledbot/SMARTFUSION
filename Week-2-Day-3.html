Data Preprocessing in Machine Learning

Data preprocessing is a crucial step in the machine learning pipeline. It ensures that the data is clean, consistent, and suitable for training algorithms, improving model accuracy and efficiency.

Data Collection

Data is gathered from various sources and formatted into structured datasets. Key sources include:

  • CSV files: Widely used format for tabular data.
  • Online datasets: Websites like Kaggle provide public datasets for various ML tasks.

Importance: The quantity and quality of the collected data directly affect model accuracy, learning, and generalization.

Data Cleaning

Cleaning the dataset involves fixing or removing incorrect, incomplete, or duplicate data entries.

  • Incomplete data: Missing fields or values.
  • Inaccurate data: Errors or incorrect measurements.
  • Inconsistent data: Different formats or mismatched categories.
Handling Missing Values

Techniques to deal with null or undefined data entries:

  • Dropping rows: Remove rows with missing values if the dataset is large.
  • Filling: Replace missing data with:
    • Mean value
    • Median value
    • Constant placeholder
  • Detection: Using functions like isnull() or isna() in Python (Pandas).
Scaling and Normalization

Brings features into the same scale to prevent dominance of larger-scale variables. Types include:

  • Standardization (Z-score): Centers around the mean with unit variance.
  • Min-Max Normalization: Scales values between 0 and 1.
Python Libraries Used in Preprocessing
  • Pandas: For data manipulation, cleaning, and analysis.
  • Numpy: For numerical operations and handling arrays.
  • Scikit-learn: For preprocessing utilities like scaling, encoding, splitting data.
