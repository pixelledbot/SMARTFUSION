Topics Covered

Data Preprocessing in Machine Learning

Data preprocessing prepares raw data for machine learning. It ensures the dataset is clean, complete, and properly formatted to improve model accuracy.

  • Cleaning data
  • Handling missing values
  • Scaling and normalization
  • Encoding categorical variables
Data Collection

Data collection involves gathering relevant information from sources like:

  • CSV and Excel files
  • Online datasets (e.g., Kaggle)
  • APIs and sensors

Good data quality and quantity are essential for accurate models.

Data Cleaning

This step corrects or removes incorrect, inconsistent, or duplicate data. Clean data ensures reliable predictions.

  • Fix typos
  • Remove duplicates
  • Standardize formats
Handling Missing Values

Missing values can affect model performance. Options include:

  • Remove rows/columns with missing data
  • Fill using mean/median (for numerical values)
  • Use most frequent value (for categorical data)
Scaling and Normalization

Feature scaling helps models like KNN, SVM, and neural networks perform better by adjusting data to a standard scale.

  • Min-Max Scaling: Scales between 0 and 1
  • Standardization: Zero mean, unit variance
Python Libraries Used in Preprocessing
  • Pandas: Data manipulation, cleaning, I/O
  • NumPy: Numerical computations
  • Scikit-learn: Preprocessing tools like scalers, encoders, imputers
